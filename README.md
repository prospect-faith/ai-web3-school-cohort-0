# AI x Web3 School Learning Repo

个人 AI x Web3 School 学习仓库，用来沉淀每日打卡、任务记录、实验原型、Handbook 反馈和 Hackathon proof-of-work。

This repository tracks my AI x Web3 School learning journey: daily notes, task logs, experiments, Handbook feedback, and Hackathon proof-of-work.

## Learning Context / 学习背景

- Program / 项目: AI x Web3 School
- Profile / 学员画像: beginner-friendly path, developer and Hackathon oriented
- Daily time / 每日投入: 60-90 minutes
- Language / 输出语言: bilingual by default, Chinese-first for daily check-ins
- Handbook / 学习手册: https://aiweb3.school/zh/handbook/
- WCB learning page / 学习入口: https://web3career.build/zh/programs/AI-Web3-School#tab=learning
- Learning Agent prompt / Agent 启动 Prompt: https://aiweb3.school/learning-agent.zh.txt

## Repository Map / 仓库结构

- `profile.md`: personal learning profile and constraints
- `learning-plan.md`: 60-90 minute daily learning plan and milestone path
- `daily/`: daily check-in drafts and reflections
- `tasks/`: individual learning task notes
- `experiments/`: small technical experiments and prototype logs
- `handbook-feedback/`: feedback drafts for Handbook improvements
- `hackathon/`: product ideas, prototype notes, demo plans, and pitch material
- `submissions/`: final links or artifacts submitted to course platforms
- `templates/`: reusable daily note and task note templates

## Daily Flow / 每日流程

1. Open the WCB learning page and Handbook.
2. Copy `templates/daily-note.md` into `daily/YYYY-MM-DD.md`.
3. Pick one minimum task, one recommended task, and optionally one challenge task.
4. Record what I learned, what I built, blockers, and one next step.
5. Draft the WCB check-in text locally, then submit manually on the platform.
6. If the Handbook can be improved, create a feedback note in `handbook-feedback/`.
