# Learning Profile / 学习画像

## Snapshot / 当前设定

- Program / 项目: AI x Web3 School
- Track / 方向: developer and Hackathon oriented
- Starting point / 起点: beginner-friendly path
- Daily capacity / 每日时间: 60-90 minutes
- Language / 输出语言: bilingual, Chinese-first for daily check-ins
- Repository / 仓库: public learning repository
- Reminder setting / 提醒设置: no automation for now

## Goals / 学习目标

1. Build shared vocabulary across AI, Agents, Web3, wallets, permissions, and on-chain interactions.
2. Learn by producing visible proof-of-work: notes, small experiments, task logs, and prototype artifacts.
3. Convert learning into a small Hackathon-ready demo idea with a clear user problem, runnable prototype, and concise pitch.
4. Give useful Handbook feedback when examples, definitions, links, or learner flow can be improved.

## Constraints / 约束

- Prefer 60-90 minute learning blocks.
- Keep each day shippable: one small artifact is better than a large unfinished plan.
- Keep all public content safe: no secrets, private wallet material, or sensitive personal data.
- Use WCB as the source of truth for daily course tasks after logging in.

## Working Style / 学习方式

- Start with the minimum path, then expand if time allows.
- Record questions and blockers quickly instead of silently getting stuck.
- Treat every experiment as proof-of-work, even if it is small.
- Keep feedback specific: quote the page link, describe the issue, suggest the fix.
